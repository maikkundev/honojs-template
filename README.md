# HonoJS Template

## A simple HonoJS with Prisma (+ more) template for building APIs

### Dependencies used

- `Hono`: Web Framework
- `Prisma`: ORM
- `zod`: Data validation
- `Vitest`: Writing tests
- `Biome.js`: Formatting and linting
- `commitlint`: Lint commit messages
- `lintstaged`: Lints staged files
- `husky`: Git hook manager
