# HonoJS Template

## A simple HonoJS with Prisma (+ more) template for building APIs

### Dependencies used

- `Deno 2`: Runtime
- `Hono`: Web Framework
- `Prisma`: ORM
- `zod`: Data validation
- `Deno Test`: Writing tests
- `Deno fmt`: Formatting
- `Deno lint`: Linting
- `commitlint`: Lint commit messages (currently not working :c)
- `lintstaged`: Lints staged files
- `deno_hooks`: Git hook manager
